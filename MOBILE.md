# モバイル操作ガイド 📱

Figma Clone はモバイルフレンドリーに設計されており、iPhone や Android のブラウザで PC と同等の機能を利用できます。

## 🎯 対応ブラウザ

- **iOS**: Safari 14+
- **Android**: Chrome 90+
- **その他**: Firefox Mobile, Edge Mobile

## ✨ タッチジェスチャー

### 基本操作

| ジェスチャー | アクション | 説明 |
|------------|----------|------|
| 👆 **タップ** | オブジェクト選択 | 選択ツール時に有効 |
| 👆👆 **ダブルタップ** | ズーム切り替え | 50% ⇔ 100% で切り替え |
| 👆⏱️ **ロングプレス** | コンテキストメニュー | 500ms長押しでメニュー表示 |
| 🤏 **ピンチイン** | ズームアウト | 2本指を近づける（10%まで） |
| 🖐️ **ピンチアウト** | ズームイン | 2本指を広げる（200%まで） |
| 👉 **ドラッグ** | オブジェクト移動 | 選択ツールで移動可能 |

### ジェスチャーの詳細

#### ダブルタップでズーム
```
1回目のタップ → 300ms以内に2回目のタップ
├─ 現在100%以上: 50%にズームアウト
└─ 現在100%未満: 100%にズームイン
```

#### ロングプレスでコンテキストメニュー
```
タップ開始 → 500ms保持 → メニュー表示
├─ コピー (選択中のオブジェクト)
├─ ペースト (クリップボードにある場合)
├─ 複製 (選択中のオブジェクト)
├─ 削除 (選択中のオブジェクト)
├─ ロック/ロック解除
├─ 最前面/最背面へ移動
└─ グループ化/グループ解除
```

#### ピンチズーム
```
2本指タッチ → 距離を変更
├─ 指を広げる: ズームイン (最大200%)
└─ 指を近づける: ズームアウト (最小10%)
```

## 🛠️ ツールの使い方

### 選択ツール（V）
- **タップ**: オブジェクトを選択
- **ドラッグ**: 選択したオブジェクトを移動
- **角のハンドル**: リサイズ（ドラッグ）

### 矩形ツール（R）
1. ツールバーの矩形アイコンをタップ
2. キャンバスをドラッグして矩形を作成
3. 自動的に選択ツールに戻る

### 円ツール（O）
1. ツールバーの円アイコンをタップ
2. キャンバスをドラッグして円を作成
3. 自動的に選択ツールに戻る

### 線ツール（L）
1. ツールバーの線アイコンをタップ
2. 開始点から終了点までドラッグ
3. 自動的に選択ツールに戻る

### 矢印ツール（A）
1. ツールバーの矢印アイコンをタップ
2. 開始点から終了点までドラッグ
3. 矢印付きの線が作成される

### テキストツール（T）
1. ツールバーのテキストアイコンをタップ
2. テキストを配置したい場所をタップ
3. キーボードで文字を入力
4. 選択ツールで移動・リサイズ可能

### 鉛筆ツール（P）
1. ツールバーの鉛筆アイコンをタップ
2. 自由に描画
3. ダークモードでは白、ライトモードでは黒で描画

## 📐 レスポンシブレイアウト

デバイスのサイズに応じて UI が自動調整されます。

### モバイル（< 640px）
```
┌─────────────────┐
│   Toolbar       │
├─────────────────┤
│                 │
│     Canvas      │
│                 │
└─────────────────┘
```
- キャンバスのみ表示
- ツールバーは簡易版
- Export/Import ボタン非表示
- ALL Reset ボタン非表示

### タブレット（640px - 768px）
```
┌─────────────────┐
│   Toolbar       │
├─────────────────┤
│                 │
│     Canvas      │
│                 │
└─────────────────┘
```
- Export/Import ボタン表示
- ALL Reset ボタン表示

### タブレット横向き（768px - 1024px）
```
┌──────┬──────────┐
│ Tool │          │
├──────┤  Canvas  │
│Layer │          │
│Panel │          │
└──────┴──────────┘
```
- LayersPanel 表示
- ページ管理機能利用可能

### デスクトップ（≥ 1024px）
```
┌──────┬──────────┬─────┐
│ Tool │          │Props│
├──────┤  Canvas  │Panel│
│Layer │          │     │
│Panel │          │     │
└──────┴──────────┴─────┘
```
- 全パネル表示
- プロパティパネルで色・サイズ調整可能

## 💡 モバイルでの便利なヒント

### 1. 横向き表示がおすすめ 📱→📲
デバイスを横向きにすると作業スペースが広くなります。

### 2. ヘルプボタンを活用 ❓
右上の`?`ボタンでいつでもヘルプを確認できます。

### 3. 自動保存 💾
作業内容はブラウザに自動保存されます。
- ページをリロードしても復元
- ブラウザのキャッシュをクリアすると削除されます

### 4. ダークモード 🌙
目に優しいダークモードに切り替え可能：
- 右上の月/太陽アイコンをタップ
- 設定は自動保存されます

### 5. マルチページ機能 📄
複数のページで作業を整理できます（768px以上）：
- LayersPanel 上部の「ページ」セクション
- `+`ボタンで新規ページ追加
- ページ名をタップで切り替え

### 6. ズームのコツ 🔍
- **素早くズーム**: ダブルタップ
- **細かい調整**: ピンチズーム
- **全体を見る**: ピンチアウトで10%まで縮小

### 7. オブジェクトの編集 ✏️
選択中のオブジェクトをロングプレスすると：
- コピー・ペースト
- 複製
- グループ化
- レイヤー順序の変更
など、様々な操作が可能

## 🚀 パフォーマンス最適化

### 推奨事項
- ✅ オブジェクト数は50個以下が推奨
- ✅ 不要なオブジェクトは削除
- ✅ 複雑な図形は画像として扱う
- ✅ 定期的にページをリロード

### 動作が重い場合
1. **ブラウザのタブを閉じる** - 他のタブを閉じてメモリを解放
2. **キャッシュをクリア** - 設定 → Safari → 履歴とWebサイトデータを消去
3. **ページをリロード** - 画面を更新
4. **オブジェクトを削減** - 不要なオブジェクトを削除

## 🔧 トラブルシューティング

### タッチが反応しない
- ページをリロード
- ブラウザを再起動
- iOS/Androidを最新版に更新

### ズームができない
- ブラウザのズームを無効化している場合があります
- Safari の設定を確認

### オブジェクトが選択できない
- 選択ツールが有効か確認
- オブジェクトがロックされていないか確認
- LayersPanel でレイヤーの目アイコンを確認

### データが消えた
- ブラウザのキャッシュをクリアしていませんか？
- プライベートブラウズモードで開いていませんか？
- 同じブラウザ・同じドメインで開いてください

## 🎨 推奨ワークフロー

### 1. 新規プロジェクト開始
```
1. ブラウザでアクセス
2. デバイスを横向きに
3. ヘルプボタンで操作を確認
4. ツールを選択して描画開始
```

### 2. 編集作業
```
1. 選択ツールでオブジェクト選択
2. ロングプレスでコンテキストメニュー
3. コピー・ペーストで効率化
4. ピンチズームで細かい調整
```

### 3. 整理・保存
```
1. グループ化で整理
2. ページ機能で分類
3. 自動保存を確認
4. スクリーンショットでバックアップ
```
